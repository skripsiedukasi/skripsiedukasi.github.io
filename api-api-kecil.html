15. Bagaimana cara mengatasi sistem pengapian pada motor tidak berfungsi (tidak ada api atau api tidak stabil)?


Jawaban & Penjelasan:

         Penyebab Pengapian Motor dan Cara Mengatasinya

                 

1. Kerusakan Aki Motor Injeksi

untuk tahu apakah sumber masalahnya berasa dari aki, Anda bisa mengecek bagian indikator pada stang motor. Indikator aki ini akan menyala saat kunci kontak berada dalam posisi on.

Anda tidak perlu mengecek perangkat secara langsung, karena aki sudah pasti habis dayanya atau rusak. Sedangkan motor dengan sistem karburator tidak akan berpengaruh jika komponen aki mengalami kerusakan.

Jadi masalahnya bukan berasal dari aki jika Anda menggunakan motor jenis karburator. Untuk mengatasi masalah ini hanya bisa dilakukan dengan ganti aki yang baru.

   2. Bahan Bakar Habis

Anda perlu mengingat-ingat kapan terakhir mengisi bensin, jika tidak bisa kehabisan di jalan. Perlu membuka tutup tangki bensin setiap hendak berkendara untuk memastikan bahwa bahan bakar masih pada.

Jika tidak Anda wajib membeli bensin terlebih dahulu sebelum berkendara. Dengan begitu pengapian pada ruang bakar akan tetap terjaga.

   3. Tidak Ada Percikan Api pada Busi

Busi juga menjadi salah satu komponen paling sering menjadi biang kerok macetnya pengapian pada sepeda motor. Fungsi utama busi adalah untuk merubah tegangan listrik dari koil menjadi percikan api. Sehingga percikan api ini akan menciptakan pengapian.

Biasanya pengendara akan mengecek kualitas busi ketika motor mogok. Caranya adalah dengan melepas busi dari rumahnya. Setelah itu masukkan kabel busi dan tempelkan ke blok mesin.

Nyalakan motor dengan kick starter dan cek apakah kabel busi mengeluarkan percikan api. Jika busi sudah tidak menciptakan percikan api, Anda tidak bisa memperbaikinya. Cara satu-satunya untuk mengatasi masalah ini adalah dengan mengganti busi.

Namun jika ternyata ketika Anda mencoba masih keluar percikan api yang besar dan bernyala biru maka kerja busi masih bagus.

   4. Sekring yang Putus

Oleh karena itulah Anda perlu mengecek apakah komponen sekring motor masih bekerja dengan baik.

Untuk tahu apakah sekring putus, Anda hanya bisa mencoba menggunakan sekring standar. Jika kendaraan tetap bisa dinyalakan, maka sumber penyebab ada di sekring yang putus.

Kerusakan yang diakibatkan putusnya sekring motor hanya bisa diganti dengan yang baru.

   5. Kerusakan pada Koil

Oleh karena itu Anda perlu mengecek apakah koil pada sistem pengapian mengalami kerusakan atau kabel koil terputus. Jika memang ada kerusakan, cara memperbaiki hanya dengan mengganti komponennya yang baru.

   6. Soket Rumah Kontak Rusak

Pertama Anda harus mengecek apakah kabel soket kendor, jika iya cukup dengan mengencangkannya saja.

Kemudian untuk mengetahui apakah soket rumah kontak masih bekerja, Anda bisa memanfaatkan tespen. Coba tekan dengan tespen pada bagian soket apakah terdapat arus listrik, jika tidak maka Anda harus mengganti soket yang baru.

Masih masalah pada rumah kontak, penyebabnya mungkin karena Anda memasang kunci rahasia sepeda motor. Anda yang membeli motor second harus tahu posisi dari kunci rahasia ini.

Biasanya terdapat di bagian yang tersembunyi. Kunci rahasia yang terkena air, debu, dan kotoran lainnya ini bisa saja rusak sehingga akan memutus arus listrik. dan pengapian pun tidak akan terjadi.  

   7. Rusaknya Stop Engine

Cara mengatasi pertama adalah dengan membuka penutup ECO kemudian membersihkannya. Nyalakan motor, jika tidak menyala juga artinya harus mengganti sistem ECO dengan yang baru.

   8. Rusaknya CDI

Komponen ini merupakan komponen dalam sistem pengapian yang fungsinya adalah untuk mengatur percikan api busi akan digunakan pada bahan bakar yang sudah dipadatkan piston. Apabila CDI tidak bekerja atau rusak, maka sistem pengapian tidak akan terjadi.

Penting sekali untuk komponen ini tetap terus terjaga, karena jika sudah rusak Anda hanya bisa menggantinya dengan yang baru.

Apabila Anda sudah mengecek sendiri satu per satu bagian tersebut namun masih saja pengapian tidak terjadi, maka segera bawa kendaraan ke bengkel resmi


16. carilah penjelasan proses terjadinya gunung api perisai,gunung api kubah,gunung api camparan,gunung api kodera,dan gunung api kerucut barca


Ditinjau dari bentuknya gunung api cuma ada 3 jenis: perisai, maar/kaldera, dan kerucut/strato. Proses terbentuknya seperti berikut.
1. Gunung api perisai: gunung api tipe ini erupsinya cenderung efusif (lelehan) cair secara terus menerus, sangat jarang ada letusan eksplosif/ledakan. Akibatnya, lelehan material dari kawah menumpuk di lereng/badan gunung hingga bentuknya seperti perisai. Contohnya gunung-gunung di Hawaii.
2. Gunung api maar/kaldera: gunung api ini pernah mengalami satu erupsi ledakan/eksplosif yang dahsyat, hingga bagian gunung api tersebut ikut terlempar bersama ledakan material gunung api tersebut. Contohnya adalah Gunung Kelud (pas dulu ada danaunya), Gunung Toba (sekarang jadi danau).
3. Gunung api strato/kerucut: erupsinya selang-seling dari efusif kadang-kadang jadi eksplosif. Contohnya Gunung Merapi.

1). Gunung api Perisai/Prisma
Gunung api perisai berciri lerengnya agak landai berbentuk perisai. Gunung api ini hanya terdiri dari lapisan-lapisan lava saja, karena lava yang keluar dari gunung api hanya berupa lava yang cair sekali, sehingga dapat mengalir jauh menuruni lereng, kemudian mengalami pembekuan. Gunung api perisai terdapat di Kepulauan Hawai yaitu Gunung Mauna Loa dan Gunung Kilauea. Di Indonesia tidak ditemukan jenis gunung api perisai.
2). Gunung api strato
Gunung api strato berciri bentuknya seperti kerucut. Strato artinya lapisan, oleh karena badan gunung api ini terdiri dari lapisan lapisan lava yang bercampur dengan hasil-hasil vulkanis lainnya seperti debu, pasir, kerikil, dan bom. Sebagian besar gunung api di Indonesia tergolong bentuk gunung api strato.

Kata maar berasal dan bahasa Jerman yang artinya “kawah”. Maar terjadi karena peletusan gunung api itu hanya terjadi satu kali saja. Setelah itu kegiatan vulkanis berhenti sama sekali.
