15. Jelaskan apakah yang dimaksud dengan protokol pada penulisan alamat website ?


Penjelasan:

Protokol Internet

(Inggris Internet Protocol disingkat IP) adalah protokol lapisan jaringan (network layer dalam OSI Reference Model) atau protokol lapisan internetwork (internetwork layer dalam DARPA Reference Model) yang digunakan oleh protokol TCP/IP untuk melakukan pengalamatan dan routing paket data antar host-host di jaringan komputer berbasis TCP/IP. Versi IP yang banyak digunakan adalah IP versi 4 (IPv4) yang didefinisikan pada RFC 791 dan dipublikasikan pada tahun 1981, tetapi akan digantikan oleh IP versi 6 pada beberapa waktu yang akan datang.

Jawaban:

Penjelasan:

1. Website adalah sebuah platform yang menyediakan informasi, konten, dan layanan yang dapat diakses melalui internet. Website dapat berupa halaman web statis atau dinamis yang dapat diakses melalui alamat URL.

2. Kegunaan website yang anda dapatkan tergantung pada jenis website yang anda buka. Misalnya, jika anda membuka website berita, anda dapat menemukan berita terbaru dan informasi lainnya. Jika anda membuka website toko online, anda dapat melihat produk yang dijual dan melakukan pembelian.

5. Cara kerja hosting website adalah dengan menyimpan semua file website di server. Server ini kemudian akan mengirimkan file tersebut kepada pengguna internet yang mengakses website tersebut. Ini memungkinkan pengguna untuk melihat dan mengakses website dari mana saja di dunia.
