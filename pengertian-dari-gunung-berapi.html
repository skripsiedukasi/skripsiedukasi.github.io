3. Jelaskan pengertian gunung berapi dan klasifikasi gunung berapi


Gunung berapi adalah Peristiwa Alam yang terjadi akibat adanya endapan magma di dalam perut bumi yang didorong keluar oleh gas yang bertekanan tinggi dari dalam perut bumi.

Klasifikasi Gunung Berapi :
1. Gunung api Tipe A : tercatat pernah mengalami erupsi magmatik sekurang-kurangnya satu kali sesudah tahun 1600.
2. Gunung api Tipe B : sesudah tahun 1600 belum tercatat lagi mengadakan erupsi magmatik namun masih memperlihatkan gejala kegiatan vulkanik seperti kegiatan solfatara.
3. Gunung api Tipe C : sejarah erupsinya tidak diketahui dalam catatan manusia, namun masih terdapat tanda-tanda kegiatan masa lampau berupa lapangan solfatara/fumarolapada tingkah lemah.

12. Pengertian Gunung Api Maar ?


Gunung Api Maar merupakan gunung api yang memiliki dapur magma yang dangkal.

Pembahasan

Vulkanisme adalah fenomena alam yang berhubungan dengan aktivitas yang terjadi pada gunung api. Vulkanisme juga merujuk pada dinamika magma yang berasal dari dapur magma lalu bergerak menuju ke permukaan bumi. Di bumi ini terdapat banyak jenis jenis gunung api baik itu berdasarkan tipe letusannya, proses terjadinya, maupun berdasarkan aktivitas vulkanisme nya.

Pengertian Gunung Api Maar ?

Penjelasan

Agar dapat mengetahui apa itu gunung Api Maar, yuk simak penjelasan berikut !

Gunung Api Maar adalah jenis gunung api yang dikelompokkan berdasarkan proses terjadinya. Gunung api ini mempunyai dapur magma yang dangkal. Penyebab terbentuknya gunung api Maar dikarenakan proses letusan. Sifat dari erupsinya yakni eksplosif, maka dari itu letusan dari gunung ini hanya terjadi satu kali. Material padat ( eflata ) merupakan bahan yang dikeluarkan saat Gunung Api Maar meletus.
