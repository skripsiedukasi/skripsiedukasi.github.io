33. Bila diketahui nilai penjualan sebesar Rp.100.000.000 dengan hpp sebesar 30% dari nilai penjualan , persedian akhir 80% dari nilai penjualan dan pembelian sebesar 50% dari persedian akhir , berapakah nilai awal dan laba kotor


Penjelasan dengan langkah-langkah:

1. Hitung Harga Pokok Penjualan (HPP):

- HPP = 30% x Nilai Penjualan = 0.3 x Rp.100.000.000 = Rp.30.000.000

2. Hitung Persediaan Akhir:

- Persediaan Akhir = 80% x Nilai Penjualan = 0.8 x Rp.100.000.000 = Rp.80.000.000

3. Hitung Pembelian:

- Pembelian = 50% x Persediaan Akhir = 0.5 x Rp.80.000.000 = Rp.40.000.000

4. Hitung Nilai Awal:

- Nilai Awal = HPP + Pembelian = Rp.30.000.000 + Rp.40.000.000 = Rp.70.000.000

5. Hitung Laba Kotor:

- Laba Kotor = Nilai Penjualan - HPP = Rp.100.000.000 - Rp.30.000.000 = Rp.70.000.000

Jadi, nilai awal adalah Rp.70.000.000 dan laba kotor adalah Rp.70.000.000.
