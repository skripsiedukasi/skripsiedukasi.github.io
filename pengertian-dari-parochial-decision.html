1. Pengertian decision support system (DSS)?


Decision Support System adalah bagian dari sistem informasi berbasis komputer (termasuk sistem berbasis pengetahuan (manajemen pengetahuan) yang dipakai untuk mendukung pengambilan keputusan dalam suatu organisasi atau perusahaan.

adalah bagian dari sistem informasi berbasis komputer (termasuk sistem berbasis menajemen umum) yang dipakai untuk mendukung pengambilan keputusan dalam suatu organisasi atau perusahaan
Dapat juga dikatakan sebagai sistem komputer yang mengolah data menjadi informasi untuk mengambil keputusan dari masalah semi-terstruktur yang spesifik.

10. Jelaskan perbedaan antara keputusan terstruktur (structured decision), keputusan setengah terstruktur (semi structured decision), dan keputusan tidak terstruktur (unstructured decision). berikan contoh untuk masing-masing jenis keputusan.


Keputusan terstruktur (structured decision) Merupakan keputusan yang dilakukan secara berulang-ulang dan bersifat rutin. Prosedur dalam pengambilan keputusan terstruktur ini sangat jelas, keputusan terstruktur dilakukan pada manajemen tingkat bawah. Contohnya adalah memilih sampul depan sebuah majalah, mengontrak manajemen tingkat senior, dan memilih proyek penelitian awal yang akan dilakukan.

Keputusan Semi Terstruktur (semi – structured decision) adalah keputusan yang sebagian dapat diprogram, sebagian berulang-ulang dan rutin dan sebagian tidak struktur. Keputusan tipe ini seringnya bersifat rumit dan membutuhkan perhitungan – perhitungan serta analisis yang terperinci.Contoh dari keputusan tipe ini adalah keputusan membeli sistem komputer yang lebih canggih, dan contoh yang lainnya adalah keputusan alokasi dana promosi.

Keputusan Tidak Terstruktur (unstructured decision) adalah keputusan yang tidak terjadi berulang – ulang dan tidak selalu terjadi. Keputusan ini terjadi di manajemen tingkat atas. Contohnya adalah memilih sampul depan sebuah majalah, mengontrak manajemen tingkat senior, dan memilih proyek penelitian awal yang akan dilakukan.

15. apa pengertian investor menerapkan konsep “think fast and decision fast”dalam konteks analisis rasio keuangan.


Pengertian investor menerapkan konsep “think fast and decision fast”dalam konteks analisis rasio keuangan yaitu investor dapat memutuskan untuk berinvestasi hanya dengan mengandalkan sebuah laporan keuangan, karena rasio keuangan mampu memberikan jawaban apa yang mereka inginkan dengan cepat.

Pembahasan

Sebuah laporan keuangan mempunyai peranan yang sangat penting bagi seorang investor untuk mengambil sebuah keputusan, karena investor tidak mempunyai cukup waktu untuk menelaah satu per satu perusahaan, jadi hal dianggap paling tepat bagi mereka dengan efisiensi waktu rasio keuangan merupakan jawabannya.

Sebuah laporan keuangan menggambarkan kondisi sebuah perusahaan, dengan laporan keuangan juga kita dapat membandingkan perusahaan mana yang paling tepat dan mempunyai masa depan yang cukup cerah jika kita berinvestasi didalamnya, karena spekulasi yang dilakukan oleh investor ini tentu mengharapkan keuntungan, oleh sebab itu para investor harus berfikir dan memutuskan dengan cepat dalam berinvestasi.

Laporan keuangan memuat semua informasi tentang sebuah perusahaan, dapat kita bandingkan kemajuan perusahaan dari waktu ke waktu berdasarkan laporan keuangan tersebut, karena sebuah laporan keuangan sudah mampu memberikan informasi yang lengkap bagi penggunanya, mulai dari besarnya harta perusahaan, kewajiban sampai dengan keuntungan dan kerugian yang mereka derita.

22. komponen bahasa pemrograman:1. Variable2. DEcision3. Iteration4. Method5. Array(silahkan berikan pengertian dan contohnya dalam bahasa C++)​


Jawaban:

Tentu, berikut adalah penjelasan singkat dan contoh penggunaan komponen bahasa pemrograman dalam bahasa C++:

1. Variable (Variabel):

- Pengertian: Variabel adalah tempat untuk menyimpan data dalam program. Ini memungkinkan Anda untuk menyimpan dan mengakses nilai-nilai yang diperlukan selama eksekusi program.

- Contoh dalam C++:

```cpp

int angka = 10;

double harga = 29.99;

std::string nama = "John";

```

2. Decision (Percabangan):

- Pengertian: Decision digunakan untuk membuat keputusan berdasarkan kondisi tertentu. Ini melibatkan pernyataan seperti `if`, `else if`, dan `else`.

- Contoh dalam C++:

```cpp

int usia = 18;

if (usia >= 18) {

std::cout << "Anda sudah dewasa." << std::endl;

} else {

std::cout << "Anda masih anak-anak." << std::endl;

}

```

3. Iteration (Pengulangan):

- Pengertian: Iteration digunakan untuk mengulang tindakan tertentu dalam program. Ini melibatkan pernyataan seperti `for` dan `while`.

- Contoh dalam C++:

```cpp

for (int i = 0; i < 5; i++) {

std::cout << "Iterasi ke-" << i << std::endl;

}

```

4. Method (Fungsi):

- Pengertian: Method adalah blok kode yang dapat dipanggil oleh program untuk menjalankan tugas tertentu. Ini digunakan untuk mengorganisasi dan merapikan kode.

- Contoh dalam C++:

```cpp

int tambah(int a, int b) {

return a + b;

}

int hasil = tambah(5, 3);

```

5. Array:

- Pengertian: Array adalah struktur data yang dapat menyimpan sejumlah elemen dengan tipe data yang sama. Elemen-elemen ini diindeks dan dapat diakses berdasarkan indeksnya.

- Contoh dalam C++:

```cpp

int angka[5] = {1, 2, 3, 4, 5};

std::cout << "Nilai pada indeks 2: " << angka[2] << std::endl;

```

Semua komponen ini adalah bagian penting dari bahasa pemrograman C++ dan digunakan untuk membangun program yang lebih kompleks.

25. Jelaskan pengertian dari variabel, decision, iteration, beserta dengan contoh code C++​


jawaban beserta penjelasan:

1. Variabel: Variabel adalah simbol yang digunakan untuk menyimpan data dalam suatu program. Variabel memiliki nama yang ditentukan oleh pengguna dan dapat diisi dengan berbagai tipe data seperti integer, float, string, dll.

if (age >= 18) {

} else {

cout << "Anda belum cukup umur" << endl;

}

3. Iteration: Iteration adalah proses pengulangan dalam suatu program. Iteration digunakan untuk mengeksekusi kode yang sama berulang kali sesuai dengan jumlah yang ditentukan.

Contoh penggunaan perulangan dalam C++ adalah perulangan for:

for (int i = 0; i < 10; i++) {

cout << i << endl;

}
