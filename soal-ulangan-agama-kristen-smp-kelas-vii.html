9. Diketahui:

m = 60 g

c = 1 kal/g°C

U = 540 kal/g

∆t dari A-B = 100-60 = 40°

• Proses A-B

=> m . c . ∆t

=> 60 . 1 . 40

=> 2400 kal

• Proses B-C

=> m . U

=> 60 . 540

=> 32.400 kal

• Proses dari A-C

=> 2400 + 32400

=> 34.800 kal (b)

10. Diketahui:

m = 4 kg

c = 4.200 J/kg°C

U = 2,27 . 10⁶ J/kg

∆t A-B = 100-10 = 90°

• Proses A-B

=> m . c . ∆t

=> 4 . 4200 . 90

=> 1.512.000 Joule ÷1000

=> 1.512 KJoule (a)

Keliling = 2πr

Luas = πr²

Keterangan :

r = jari-jari

π = 22/7 atau 3,14

Pembahasan

4. Luas bangun lingkaran

Luas lingkaran seluruhnya :

L = πr² = 3.14 x (10.5)² = 3.14 x 110.25 = 346,185 cm²

Luas ¼ lingkaran

L = ¼πr² = ¼ (3.14)(10.5)² = 86,54 cm²

Maka, luas lingkaran pada gambar adalah :

L = luas seluruh lingkaran - luas ¼ lingkaran

L = 346,185 cm² - 86,54 cm²

L = 259,645 cm²

5. Total biaya menanam rumputLuas taman = πr²= 22/7 x (28)²= 22/7 x 784= 2.464 m²

Karena biaya per m² adalah Rp7.500 maka biaya total menanam rumput adalah:

= 2.464 x 7.500

= Rp 18.480.000

Mengasihi berarti tidak suka menyakiti atau melukai hati. Mengasihi berarti tidak mengharapkan sesuatu yang buruk menimpa, terlebih berkeinginan untuk melukai atau menyakiti. Bukankah tidak sedikit orang yang enggan menerima nasihat, sekalipun hal itu menyelamatkan dan membangun? Meski demikian, jangan sampai hal ini menghentikan penyataan kasih kita. Apalagi jika kita takut menegur karena mencari aman: agar tidak diusik ketika melakukan dosa. Bukankah baik saling menegur dalam kasih? Bukan karena rasa benci, ingin menjatuhkan atau menghancurkan!
