34. membuka dokumen yang sudah ada atau tersimpan ​


Jawaban:Menemukan & membuka file

Buka File .

Di sebelah kiri, pilih tempat file disimpan.

Untuk file yang baru saja Anda gunakan, pilih Terbaru.

Untuk melihat daftar file terbaru menurut jenisnya, gunakan tombol untuk memilih jenis file tersebut.

Untuk file aplikasi Android, pilih File saya lalu File Play. Untuk menampilkan lebih banyak folder, di bagian kanan atas, pilih Lainnya  lalu Tampilkan semua folder Play.

Untuk file di cloud Anda, pilih folder, misalnya Google Drive. Anda juga dapat menambahkan sistem file cloud lain.

Temukan file dan klik dua kali untuk membukanya.

Jika Anda membuka file di perangkat penyimpanan eksternal, lepas perangkat dengan aman setelah Anda selesai dengan mengklik Keluarkan .

Tips: Di pojok layar Anda, pilih Peluncur , lalu masukkan nama file yang Anda inginkan.

Menyimpan file

Untuk menyimpan halaman, gambar, atau dokumen yang dilihat:

Tekan Ctrl + s.

Di bagian bawah, masukkan nama untuk file Anda.

Opsional: Di sebelah kiri nama file, ganti jenis file.

Di kolom sebelah kiri, pilih tempat untuk menyimpan file, seperti Google Drive lalu Drive Saya.

Pilih Simpan.

Catatan: Jika Anda menggunakan Chromebook Anda di sekolah atau kantor, Anda dapat menyimpan file dalam folder Drive bersama Anda. Pilih Google Drive lalu Drive bersama.

Mengganti nama file

Menambahkan sistem file cloud lainnya

Mengubah tempat file disimpan

Menghapus file

Penting: Sebagian besar file langsung dipindahkan ke Sampah. Untuk file yang tidak dipindahkan ke Sampah, Anda harus mengonfirmasi bahwa Anda ingin menghapusnya secara permanen.

Di pojok layar Anda, pilih Peluncur .

Buka File .

Pilih file atau folder yang ingin Anda hapus.

Pilih Sampah.

Tips: File akan dihapus permanen dari Sampah setelah 30 hari.

Untuk memulihkan file yang telah dipindahkan ke Sampah:

Di Chromebook, buka File .

Di sebelah kiri, pilih Sampah .

Pilih file yang ingin Anda pulihkan lalu Pulihkan dari Sampah.

Memasukkan file dalam zip

Untuk menggabungkan file atau folder menjadi satu file, Anda dapat memasukkannya dalam ZIP.

Di pojok layar Anda, klik Peluncur .

Buka File .

Di sebelah kiri, pilih folder atau file yang ingin dimasukkan dalam zip.
