Studi Kasus MODEL SELEKSI DECISION Buatlah Flowchartnya dari instruksi dibawah ini : Data Input KB, JB Decision / kondisi : Jika KB = "A" atau NB = "Anggur" Jika KB = "B" atau KB="a" maka : HB = 50000 KB="b" maka NB = "Belimbing" : HB = 250000 Jika KB = "C" atau KB = "c" maka : HB = 10000 NB = "Ciremai" Jika KB = "D" atau KB="d" maka NB = "Duku" : HB = 20000 Proses Disc Jika JB 10 Disc = 7500 : Proses Bayar Bayar = (JB x HB) - Disc Output KB, JB, NB, HB, Disc, Bayar​

Penjelasan:

saya tidak dapat membuat flowchart. Namun, saya dapat memberikan pseudocode untuk model seleksi decision tersebut:

Input data KB dan JB

Jika KB = "A" atau NB = "Anggur", maka HB = 50000

Jika KB = "B" atau KB = "a", maka NB = "Belimbing" dan HB = 250000

Jika KB = "C" atau KB = "c", maka NB = "Ciremai" dan HB = 10000

Bayar = (JB x HB) - Disc

Output KB, JB, NB, HB, Disc, Bayar

Dengan adanya Kespro, petani dapat memiliki akses yang lebih baik ke pasar, meningkatkan produktivitas, dan memperoleh pendapatan yang lebih tinggi. Selain itu, Kespro juga dapat membantu menciptakan nilai tambah pada produk pertanian, sehingga petani dapat lebih kompetitif di pasar global.

Namun, untuk mencapai hasil yang optimal, diperlukan kerjasama dan dukungan dari berbagai pihak, termasuk pemerintah, perguruan tinggi, sektor swasta, dan masyarakat. 31. Contoh soal mencari konsentrasi basa lemah jika diketahui ph dan kb-nya


Mapel: Kimia Semester I

Kelas: XI SMA (MIPA)

Materi: Larutan Asam Basa (BAB 5)

Kata Kunci: pH, Basa Kuat, Basa Lemah, OH-, pOH

Kode soal: 7

Kode Kategorisasi: 11.7.5
________________________________________

Pembahasan

KOH merupakan senyawa basa kuat dengan Ciri-ciri tidak memiliki kb dalam soal.
Karena Ini adalah basa kita perlu pOH sebelum pH

contoh Soal basa kuat
Hitunglah pH KOH 0,008M

Jawab

tulis reaksi ionisasi terlebih dahulu
KOH → K+ + OH-
0,008 → 0,008

Step 1 Mencari Konsentrasi OH-

[OH-] = Molaritas×Valensi Basa
[OH-] = 0,008×1
[OH-]= 0,008
[OH-] = 8×10-³

Step 2 mencari pOH

pOH = -log[OH-]
pOH= -log8×10-³
pOH= 3-log8

Step finish mencari pH

pH= pKw-pOH
pH= 14-(3-log8)
pH= 11+log8
pH= 11+log2³
pH= 11+3log2

contoh soal basa lemah
hitunglah pH NH4OH 0,01M jika diketahui Kb (10^-5)

Jawab
Step 1 menuliskan reaksi Ionisasi
