2. Menuliskan Komentar

Untuk menambahkan komentar kita hanya perlu menambahkan tanda //

public class MyClass {

 public static void main(String[] args) {

   System.out.println("Hello World"); // This is a comment

 }

}

Maka yang ditampilkan :

Hello World

   System.out.println(fullName);  

 }

}

4. Menggunakan Tipe Data

Tipe data pada setiap bahasa pemrograman apapun sangat penting untuk dipelajari. Untuk contoh penggunaan Tipe Data pada Java yaitu :

public class MyClass {

 public static void main(String[] args) {

   int myNum = 5;               // integer (whole number)

   float myFloatNum = 5.99f;    // floating point number

   char myLetter = 'D';         // character

   boolean myBool = true;       // boolean

   String myText = "Hello";     // String    

   System.out.println(myNum);

   System.out.println(myFloatNum);

   System.out.println(myLetter);

   System.out.println(myBool);

   System.out.println(myText);

 }

}

Yang ditampilkan :

5

5.99

D

true

Hello

1) basic class => beserta penjelasannya

// pembuatan class pada umumnya di awali dengan huruf kapital ( mengikuti konvensi penamaan ada umumnya

class Orang{

 // class field

 species = 'human';

 // setiap class memiliki constructor ( syntax mungkin berbeda pada setiap bahasa pemograman

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

}

// instansiasi sebuah class, class tidak dapat dipangil langsung karena dia merupakan sebuah blueprint

const polin = new Orang('polin', 19);

// output Object polin (instansi dari Orang)

console.log(polin);

2. class dengan method

class Orang{

 species = 'human';

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

 // sebuah fungsi didalam class bisa juga disebut method

 // method ini ada untuk setiap instansiasi dari class Orang

 greet() {

   return `Halo nama saya, ${this.name}`;

 }

}

const polin = new Orang('polin', 19);

// output "Halo nama saya, polin"

console.log(polin.greet());

3.class inheritance (penurunan class)

class Orang {

 species = 'human';

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

 greet() {

   return `Halo nama saya, ${this.name}`;

 }

}

// extends berarti semua method dan field dari Orang ada di Murid

class Murid extends Orang{

 perkenalanDiri() {

   return `nama saya, ${this.name} dan umur saya ${this.age}`;

 }

}

//instansiasi dari murid turunan dari orang

const popon = new Murid('popon', 19);

// output "nama saya popon dan umur saya 19"

console.log(popon.perkenalanDiri());

38. contoh class student​


Jawaban: contoh class "Student" dalam bahasa pemrograman Python:

class Student:

def __init__(self, name, age, grade):

self.name = name

self.age = age

self.grade = grade

def study(self, subject):

print(f"{self.name} is studying {subject}.")

def take_exam(self, subject):

print(f"{self.name} is taking an exam in {subject}.")

def get_grade(self):

print(f"{self.name}'s grade is {self.grade}.")

# Contoh penggunaan class Student

student1 = Student("John", 17, "A")

student1.study("Math") # Output: John is studying Math.

student1.take_exam("Science") # Output: John is taking an exam in Science.

student1.get_grade() # Output: John's grade is A.
