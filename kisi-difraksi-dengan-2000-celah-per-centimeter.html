23. Apa itu difraksi celah ganda dan celah jamak? Dan contohnya.


Sifat gelombang cahaya  

NB :  

a.Di kelas 12 hanya dibahas sifat Interferensi, difraksi dan polarisasi  

b.Persamaan yang digunakan untuk Interferensi, difraksi dan kisi difraksi semua sama :  

d sin θ =Δ m λ

d p = Δm L λ

Keterangan:  

p = jarak pita terang ke-m  

L = jarak celah ke layar

Hanya definisi ( d ) dalam persamaan memiliki pengertian yang berbeda :  

Interferensi : d = jarak antara 2 celah  

Difraksi tunggal : d = diameter celah  

Kisi Difraksi : d = jarak antar celah yang nilainya = 1/N

(1)Interferensi Young ( Celah Ganda )  

d sin θ = Δm λ

d p = Δm L λ

(2) Kisi Difraksi  

d sin θ = Δm λ

d p = Δm L λ

d = 1/N  

Untuk mempermudah dalam menentukan nilai Δm dalam penyelesaian soal, maka pada interferensi ganda maupun kisi difraksi dapat menggunakan aturan berikut :  

terang ke terang yang berdekatan (Δm = 1 )  

gelap ke gelap yang berdekatan (Δm = 1 )  

 terang ke gelap yang berdekatan (Δm = ½ )

( 3 ) Difraksi Tunggal  

d sin θ = m λ

d p = m L λ

Gelap 1 ke titik tengah terang pusat ( m = 1 )  

Gelap 2 ke titik tengah terang pusat ( m = 2 ) dst

( 4 ) Polarisasi

Polarisasi dengan absorpsi (penyerapan) selektif

-Intensitas setelah melewati polarisator:  

[tex]I_1=0,5I_o[/tex]

-Intensitas setelah melewati analisator:  

[tex]I_2=I_1cos^2\theta[/tex]

[tex]\theta[/tex] = sudut antara sumbu analisator dengan polarisator

Pembahasan  

Difraksi celah ganda adalah difraksi yang menggunakan dua celah ganda, seperti pada interferensi celah ganda,

sedangkan Difraksi celah jamak adalah difraksi yang menggunakan banyak celah , seperti pada difraksi kisi

Jawaban:

5000 garis/cm

Penjelasan:

Dik :

θ = 30°

λ = 500 nm = 5 × 10^-5 cm

m = 2

Dit : N?

Jawab

Untuk mencari N sebelumnya kita harus mencari dulu nilai d dengan menggunakan rumus :

d.sinθ = m. λ

d. sin30° = 2. 5 × 10^-5 cm

d. 0,5 = 10 × 10^-5 cm

d = 2 × 10^-4 cm = 0,0002 cm

Selanjutnya tinggal mencari nilai N

N = [tex]\frac{1}{d}[/tex]

N =  [tex]\frac{1}{0.0002}[/tex]

N = 5000 garis/cm

Diketahui

Panjang gelombang λ = 500 nm

Garis terang orde kedua atau n = 2

Sudut deviasi (penyimpangan) θ = 30°

Ditanya

Tetapan kisi N (garis/mm)

penyelesaian :

harus ada rumus= d sinθ = nλ

λ = panjang gelombang cahaya

Konversi satuan panjang gelombang cahaya

λ = 500 nm

500 * 10^ -9 m > 5 * 10^-7 m

pengerjakan : d sinθ = nλ

selanjutnya atur N sebagai subyek yang ditanyakan

N= sinθ / nλ

substitusikan

N = 500 garis/mm

m = 2

θ = 30'

Dit: λ.....?

d = 1/N

d = 1/500 mm = 2. 10^-6 m

d sin θ = mλ

λ = d sin θ/m

λ = 2.10^-6 x sin 30 / 2

λ = 2.10^-6 x 0,5 / 2

λ = 1.10^-6 / 2

λ = 0,5 x 10^-6 m

λ = 5 x 10^-7 m

atau

λ = 5000 cm
