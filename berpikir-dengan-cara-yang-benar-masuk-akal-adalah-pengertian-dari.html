38. Quizz Pakai cara dan cara pengerjaan dengan benar (Masuk akal)​


Besar < A adalah <A = 60°

___________________Pendahuluan :

Aturan cosinus pada segitiga

a² = b² + c² - 2bc Cos A

Langkahnya kita ubah dulu persamaan tersebut sehingga bisa sama dengan aturan cosinus

Diketahui :

Di ketahui segitiga ABC

Berlaku

(a + b + c) ( b + c - a).= 3bc

Besar < A adalah ....

Ditanya Besar < A adalah ?

Jawab :Jadi <A = 60°

Penyelesaian :

Di ketahui segitiga ABC

Berlaku

(a + b + c) ( b + c - a).= 3bc

ab + ac - a² + b² + bc - ab + bc + c² - ac = 3bc

- a² + b² + c² + 2bc = 3bc

b² + c² - a² = bc

b² + c² - bc = a²

Aturan cosinus pada segitiga

a² = b² + c² - 2bc Cos A

Maka

b² + c² - bc = b² + c² - 2bc Cos A

- bc = - 2bc Cos A

Cos A = -bc/-2bc

Cos A = 1/2

Jadi <A = 60°
