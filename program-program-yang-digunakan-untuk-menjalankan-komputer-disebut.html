23. program yang digunakan untuk menjalankan komputer pertama kali disebut


Program yang digunakan saat komputer berjalan pertama kali adalah Program BIOS. BIOS merupakan singkatan dari Basic Input Output System. BIOS biasanya terdapat pada motherboard.

Secara umum, motherboard terbagi menjadi beberapa jenis, yaitu

Motherboard ATXMotherboard mini ITXMotherboard micro ATXMotherboard E-ATX

Untuk bisa masuk ke dalam program BIOS, setelah komputer dinyalakan, pengguna dapat menekan tombol pada keyboard, biasanya tombol yang digunakan untuk masuk ke BIOS adalah tombol F2, F10 atau tombol ESC.

Pembahasan

BIOS merupakan sebuah software yang ada pada setiap komputer, BIOS memiliki beberapa fungsi yaitu

Melakukan pengujian pada hardware, seperti mengecek RAMMembantu proses bootingMembantu proses pengaturan dasar, seperti pengaturan tanggal komputer

Proses installasi sistem operasi pada komputer juga bergantung pada BIOS, karena dengan BIOS pengguna dapat mengatur urutan booting.

Proses booting pada komputer sendiri terbagi menjadi dua yaitu

Warm booting, merupakan proses booting yang terjadi pada saat komputer sedang digunakan, contohnya restart.Cold booting, berbeda dengan warm booting, cold booting terjadi setelah komputer tidak mendapatkan daya listrik. Contohnya booting setelah shutdown.Soft booting, booting ini hampir sama dengan warm booting, namun pada proses soft booting, proses restart dilakukan secara otomatis oleh sebuah program. Contohnya restart secara otomatis pada saat penginstallan sistem operasi.Hard booting, cara booting ini merupakan cara yang dilakukan dengan cara menekan tombol power pada komputer atau menggunakan kombinasi tombol CTRL+ALT+DEL, hard booting dilakukan karena komputer sudah tidak bisa merespon.

A. Infrastructure B. Transport C. Application D. Session E. Data Link


OSI layer yang bertanggung jawab atas pertukaran informasi antar program komputer adalah Application Layer. Open System Interconnection atau yang disebut OSI layer merupakan standar di dalam jaringan komputer untuk melakukan komunikasi data. Model OSI ini yang akan menggambarkan bagaimana berpindahnya informasi dari suatu aplikasi di komputer ke aplikasi yang ada pada komputer lain.


Pembahasan

Yang merupakan OSI layer terdiri dari

Application : layer yang bertanggungjawab atas pertukaran informasi antar program komputerPresentation : layer yang berfungsi untuk menterjemahkan data ke format yang dapat dikirim melalui jaringanSession : berfungsi sebagai pendefinisi kapan koneksi dibuat dan dihancurkanTransport : berfungsi untuk memecah data ke dalam paket -paket data yang lebih kecil.Network : berfungsi sebagai penentu alamat dan pembuat header untuk paket - paket dataData Link : berfungsi untuk mengelompokkan bit-bit data menjadi framePhysical : berfungsi untuk menentukan media tyang digunakan dalam transmisi jaringan

dan terbagi ke dalam dua kategori, yaitu "upper layer" dan "lower layer". Upper layer tediri dari session, presentation dan application. Upper layer adalah layer yang berhubungan dengan aplikasi. Di mana semakin tinggi atau ke atas maka semakin dekat dengan aplikasi.

Sedangkan lower layer terdiri dari physical, data link, network dan transport. Lower layer yang akan berhubungan dengan perangkat jaringan komputer. Di sini lah intisari komunikasi data pada jaringan terjadi, di mana semakin rendah atau ke bawah semakin berhubungan dengan perangkat fisik.
