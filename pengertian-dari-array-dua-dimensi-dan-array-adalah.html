6. perbedaan array dua dimensi dan multidimensi


Pengertian Array Dua Dimensi
Array dua dimensi dapat dipandang sebagai gabungan array satu dimensi.
Array dua dimensi merupakan array yang terdiri dari m buah baris dan n buah kolom.
Bentuknya dapat berupa matriks atau tabel.
Deklarasi array :
Tipe_array nama_array[baris][kolom];
Contoh :
Int X[3][4];
Pengertian Array Multidimensi
Array multi-dimensi merupakan array yang mempunyai ukuran lebih dari dua.
Bentuk pendeklarasian array sama saja dengan array dimensi satu maupun array dimensi dua.
Bentuk umumnya yaitu :
tipe_array nama_array[ukuran1][ukuran2]…[ukuranN];
Contoh :
float X[2][4][3];

13. perbedaan array satu dimensi, dua dimensi dan tiga dimensi​


Berikut adalah perbedaan dari array satu, dua, dan tiga dimensi:

Satu dimensi: jenis elemen array yang hanya terdiri dari satu kolom atau baris. Array satu dimensi hanya bisa diakses dengan program yang memiliki indeks tertentu.Dua dimensi: jenis elemen array yang terdiri dari 2 bentuk index array sehingga memiliki dua atau lebih kolom dengan banyak baris. Array dua dimensi akan digambarkan dalam bentuk matriks atau grid.Tiga dimensi: jenis elemen array yang memiliki kolom, baris, dan juga berukuran lebih banyak ketimbang array dua dimensi. Penomoran index pada array tiga dimensi menggunakan 3 buah angka.

Pembahasan

Array adalah salah satu istilah yang digunakan dalam dunia data di mana array adalah suatu larik yang berisikan data-data yang memiliki tipe atau jenis serupa. Array biasanya digunakan untuk menggambarkan bagaiman suatu data disimpan. Tujuan utama dari penggunaan array pada kumpulan data kita adalah untuk menyederhanakan cara mengumpulkan atau mengkategorisasikan data-data kita. Maka dari itu, penggunaan array bisa meningkatkan efektivitas dan juga efisiensi kita ketika mengumpulkan suatu data karena bisa menjadikan beberapa data menjadi satu larik saja. Ada beberapa jenis array yang bisa kita gunakan, yaitu array satu dimensi, dua dimensi, dan tiga dimensi.

20. Apakah perbedaan antara Array satu dimensi dan Array multidimensi? Jelaskan!​


Jawaban:

Perbedaannya

* Array Satu Dimensi : elemen array yang hanya memiliki satu buah baris atau kolom dan hanya dapat diakses oleh program menggunakan suatu indeks tertentu.
