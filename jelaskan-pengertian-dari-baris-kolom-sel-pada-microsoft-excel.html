12. perpotongan kolom dan baris pada Microsoft Excel ​


Perpotongan kolom dan baris pada Microsoft Excel adalah interseksi antara satu kolom dan satu baris pada seleksi spreadsheet. Ini merupakan lokasi spesifik pada spreadsheet tempat data dapat ditempatkan atau ditampilkan. Perpotongan kolom dan baris memiliki nomor unik yang terdiri dari nama kolom (seperti A, B, C, dst.) dan nomor baris (seperti 1, 2, 3, dst.). Misalnya, perpotongan antara kolom A dan baris 1 adalah A1, dan perpotongan antara kolom B dan baris 2 adalah B2. Data dapat dimasukkan ke dalam perpotongan kolom dan baris untuk menyimpan dan mengelola informasi dalam spreadsheet.

16. Jelaskan langkah-langkah untuk mengatur kolom dan baris pada Microsoft Excel​


Jawaban: Mengatur kolom di Ms. Excel 2007:

1. Klik sel yang akan disetting tinggi baris dan lebar kolomnya

2. Klik menu ribbon Home

3. Klik menu Format

4. Klik Column Width dan isikan sesuai selera untuk menentukan lebar kolom

5. Klik tombol OK

Mengatur tinggi baris di Ms. Excel 2007:

1. Klik sel yang akan disetting tinggi baris dan lebar kolomnya

2. Klik menu ribbon Home

3. Klik menu Format

4. Klik Row Height dan isikan sesuai selera untuk menentukan tinggi baris

5. Klik tombol OK
