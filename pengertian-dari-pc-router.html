11. Diketahui IP Address PC Server yaitu 192.168.1.1/25, tersambungke 1 buah router A dengan IP Address 192.168.25.1, Router A tersambungdengan Router B yang mempunyai IP Address 192.168.26.1, Switch A, dan Access Point A. Router A, Switch A dan Access Point A tersambung masing-masing 1 PC 1. Berapakah IP Address PC Yang tersambungoleh Switch A... a. 192.168.1.2 b. 192.168.100.1 C. 192.168.25.10 d. 192.168.1.10 192.168.100.2 lah In Addros RC Yang tersambungoleh Router B e. O​


Untuk menentukan IP Address PC yang tersambung oleh Switch A, kita perlu melihat subnet yang digunakan oleh perangkat dalam jaringan ini. Berdasarkan informasi yang diberikan, kita tahu bahwa IP Address PC Server adalah 192.168.1.1/25. Notasi /25 menunjukkan bahwa subnet mask adalah 255.255.255.128.

Kemudian, kita tahu bahwa Router A memiliki IP Address 192.168.25.1. Ini menunjukkan bahwa Router A berada dalam subnet yang berbeda dari PC Server. Router A, Switch A, dan Access Point A masing-masing tersambung ke 1 PC. Dikarenakan Switch A berada dalam jaringan yang sama dengan Router A, kita bisa berasumsi bahwa PC yang tersambung ke Switch A memiliki IP Address dalam subnet yang sama dengan Router A.

Dari pilihan yang diberikan, hanya IP Address 192.168.25.10 (pilihan C) yang berada dalam subnet yang sama dengan Router A (192.168.25.1). Oleh karena itu, IP Address PC yang tersambung oleh Switch A adalah 192.168.25.10.

Dynamic Host Configuration Protocol (DHCP) merupakan service yang memungkinkan perangkat dapat mendistribusikan/assign IP Address secara otomatis pada host dalam sebuah jaringan. Cara kerjanya, DHCP Server akan memberikan response terhadap request yang dikirimkan oleh DHCP Client.

Selain IP Address, DHCP juga mampu mendistribusikan informasi netmask, Default gateway, Konfigurasi DNS dan NTP Server serta masih banyak lagi custom option (tergantung apakah DHCP client bisa support).

Mikrotik dapat digunakan sebagai DHCP Server maupun DHCP Client atau keduanya secara bersamaan. Sebagai contoh, misalnya kita berlangganan internet dari ISP A.
