Penjelasan:

PEMBAHASAN

Skala dalam matematika dapat disingkat menjadi S, Jarak Sebenarnya dalam matematika dapat disingkat menjadi JS, dan Jarak pada Peta dalam matematika dapat disingkat menjadi JP. Selain itu, ada juga rumus-rumus mencari Akal, JP, dan JS adalah sebagai berikut :

Rumus mencari Skala

[tex]Skala \: = \: \frac{jarak \: pada \: peta \: }{jarak \: sebenarnya} [/tex]

Rumus mencari Jarak pada Peta

Jarak pada Peta = Skala × Jarak Sebenarnya

Rumus mencari Jarak Sebenarnya

[tex]jarak \: sebenarnya \: = \: \frac{jarak \: pada \: peta}{skala} [/tex]

Contoh Soal 1

Pulau Y dan pulau Z jaraknya 250 km. Jarak tersebut akan digambar dengan ukuran 5 cm pada sebuah buku gambar. Skala yang digunakan adalah ....

Jawaban :

250 km = 25.000.000

[tex]Skala \: = \: \frac{jarak \: pada \: peta \: }{jarak \: sebenarnya} [/tex]

[tex]Skala \: = \: \frac{ \: 5 \: cm \: }{25.000.000} \: = \: \frac{1}{5.000.00} \: = \: 1 \: : \: 5.000.000[/tex]

Contoh Soal 2

Dalam suatu peta, Kota P dan Kota Q berjarak 3 cm. Jika skala yang digunakan adalah 1 : 1.500.000, berapakah jarak sebenarnya kedua kota itu...

[tex]jarak \: sebenarnya \: = \: \frac{jarak \: pada \: peta}{skala} [/tex]

[tex]jarak \: sebenarnya \: = \: 3 \: \div \: \frac{1}{1.500.000} \\ jarak \: sebenarnya \: = \: 3 \: \times \: \frac{1.500.000}{1} \\ jarak \: sebenarnya \: = \: \frac{4.500.000}{1} \\ jarak \: sebenarnya \: \: = \: 4.500.000 \: cm \: = \: 45 \: km[/tex]

37. Segitika siku2 mempunyai tinggi 6mm dan alas 9mm, dengan sudut 30 derajat, cari sisi miring karena tidak diketahui


Jawaban:

15mm

Penjelasan dengan langkah-langkah:

diketahui :

segitiga siku-siku, sudut 30°

tinggi = a = 6mm

alas = b = 9mm

sisi miring = c...?

dengan rumus phytagoras

c =

[tex]c \: = \sqrt{ {a}^{2} } + \sqrt{ {b}^{2} } [/tex]

[tex]c = \sqrt{ {6}^{2} } + \sqrt{ {9 }^{2} } \\ c = \sqrt{36 + 81 } \\ c = \sqrt{225} \\ c = 15[/tex]

jadi, sisi miring segitiga siku-siku tersebut = 15mm
