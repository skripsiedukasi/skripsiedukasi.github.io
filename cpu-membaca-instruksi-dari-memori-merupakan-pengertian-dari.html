8. Jelaskan Alur instruksi di dalam cpu?


jawab:

Pada saat suatu program akan dieksekusi atau diproses, maka isi dari sebuah program tersebut yang berada di dalam harddisk akan di diambil dan di masukkan ke RAM.

Control Unit membedakan instruksi dan data, instruksi ditempatkan ke Program-Penyimpanan sedangkan data ditempatkan di Working-Storage. Selanjutnya instruksi dan data itu di ambil oleh Control Unit di simpan di Register.

Bila instruksi merupakan perhitungan logika atau aritmatika, maka ia dibawa ke ALU untuk diproses. Hasilnya akan ditampung di Accumulator. Dan Control Unit akan mengambilnya dan membawanya kembali ke RAM yang kemudian di tampilkan hasilnya ke alat output, seperti Monitor.

13. sebutkan dan jelaskan tiga lokasi memori yang ada di dalam cpu


Lokasi memori berada pada 3 lokasi, yaitu:
*.Memori Local atau sering disebut dengan register. Built-in berada dalam CPU, diperlukan untuk semua kegitan CPU.
*.Memori Internal atau sering disebut dengan memory primer atau memory utama. Berada diluar CPU bersifat internal pada system computer, diperlukan oleh CPU dalam proses eksekusi (operasi) program sehingga dapat diakses secara langsung oleh CPU tanpa melalui perantara.
*.Memori Eksternal atau sering disebut dengan memori sekunder. Bersifat eksternal dan berada di luar CPU, diperlukan dlam menyimpan data atau instruksi secara permanen, terdiri atas perangkat storage seperti: disk, pita magnetik, dll.

27. Bagaimana CPU bekerja dan mengoordinasi semua sumber daya dan memori?


jawaban :

1 Memori komputer mengacu pada area tempat data dan program disimpan. Memori bukan bagian dari CPU, tetapi CPU harus berinteraksi erat dengannya. Ada dua jenis memori komputer: primer, atau utama, dan sekunder.

2 CPU sangat bergantung pada memori utama untuk menyimpan instruksi program dan data instruksi beroperasi. Memori utama bersifat sementara dan hanya menyimpan instruksi dan data untuk suatu program saat program sedang dijalankan.

Jawaban & Penjelasan:

CU-Control Unit- mengkoordinasikan, mengelola, dan mengontrol semua bagian komputer.

CPU Unit Pemrosesan Pusat

CPU adalah perangkat pemrosesan utama sistem komputer yang memproses data input dan mengubahnya menjadi informasi yang bermakna yang ditampilkan pada monitor dalam bentuk output.

CPU mengontrol dan mengelola semua aktivitas komputer sehingga disebut juga otak komputer. CPU memiliki tiga unit utama sebagai unit kontrol, unit aritmatika dan logika, dan unit memori.

Unit Kontrol (CU):

Unit Kontrol mengontrol dan mengelola semua aktivitas komputer. Ini mengkoordinasikan semua unit komputer.

Unit Aritmatika dan Logika (ALU):

ALU melakukan semua perhitungan aritmatika dan logika seperti penjumlahan, pengurangan, perkalian dan pembagian. Selain itu, ia melakukan perhitungan logis seperti membandingkan, memilih, dll.

Unit Memori:

Unit memori menyimpan data dan instruksi komputer. Ini menyimpan data keluaran

29. Jelaskan pengertian dan Perbedaan Memori ROM dan RAM pada CPU


Jawaban:

RAM merupakan jenis penyimpanan sementara, karena data hanya berjalan saat daya komputer hidup.

Sedangkan ROM merupakan memori permanen yang menyimpan data dalam jangka lama. RAM merupakan memori yang bersifat menguap.

Perbedaaan

Dari Jenis kerja

Data yang disimpan dalam RAM dapat diambil dan diubah.

Data yang disimpan dalam ROM hanya dapat dibaca.

Dari Segi Kecepatan

RAM => Ini adalah memori berkecepatan tinggi.

ROM => Ini jauh lebih lambat dari pada RAM.
