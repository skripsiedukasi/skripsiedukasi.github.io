13. A Apa yang disebut bilangan? B. Sebutkan macam ? bilangan dan contohnya ?​


Jawaban:

A.) Bilangan adalah suatu konsep matematika yang digunakan untuk pencacahan dan pengukuran

B.) Bilangan Nol

Bilangan nol adalah bilangan nol itu sendiri (0) Contoh: N = { 0 }

Bilangan Pecahan

Contoh: H = { ⅓, ⅔, ⅛, ⅝, ….. } Keterangan tambahan: 4/2 = 2, berarti 4/2 bukan termasuk pecahan.

Bilangan Negatif

Bilangan negatif adalah macam-macam bilangan bernilai negatif. Contoh: N = { -3, -5, ¼, …. } Keterangn tambahan: -2/-3 = ⅔, berarti -2/-3 bukan termasuk bilangan negatif.

Bilangan Positif

Bilangan positif adalah macam-macam bilangan yang bernilai positif selain nol. Contoh: P = { 2, 3, 4, ¼, …. }

Bilangan asli

adalah himpunan bulat positif yang bukan nol {1, 2, 3, 4, ...}, hal tersebut menurut definisi matematikawan tradisional.

Namun bagi para logikawan dan ilmuwah komputer, bilangan asli adalah himpunan nol dan bagian bulat positif yakni {0, 1, 2, 3, ...}.

Bilangan Prima

Sepuluh bilangan prima adalah 2, 3, 5, 7, 11, 13, 17, 23 dan juga 29. Apabila bilangan lebih besar dari satu dan bukan bilangan prima maka dapat dikatakan sebagai bilangan komposit.

Bilangan Cacah

Macam-macam bilangan dalam matematika, salah satunya adalah bilangan cacah. Bilangan cacah adalah himpunan bilangan bulat yang tidak negatif, yaitu {0, 1, 2, 3 ...}.

Bilangan Bulat

Bilangan bulat positif adalah bilangan yang memiliki nilai positif mulai dari angka 1 ke atas dan seterusnya (1,2,3,4 dan seterusnya).

Sedangkan bilangan bulat negatif adalah bilang yang dimulai dari bilangan negatif 1 ke bawah dan juga seterusnya. Contohnya adalah seperti -1,-2,-3,-4 dan seterusnya.

Bilangan Rasional

Macam-macam bilangan rasional adalah bilangan yang bisa dinyatakan sebagai p/q di mana, p,q ϵ bulat dan q ≠ 0. Dapat pula dinyatakan sebagai suatu bilangan desimal dengan cara berulang-ulang. Bilangan rasional memiliki batasan yakni mulai dari selangga (-∞, ∞).

Bilangan rasional adalah bilangan yang dapat dinyatakan sebagai a/b di mana, a, b bilangan bulat dan b tidak sama dengan 0. Bilangan rasional di dalamnya sudah mencakup bilangan lain sepeti bilangan asli, bilangan cacah, bilangan bulat, dan juga bilangan prima.

Bilangan Irrasional

Contohnya adalah π, dan bilangan e. Bilangan π sebetulnya tidak tepat, yaitu kurang lebih 3.14, tetapi = 3,1415926535.... atau = 3,14159 26535 89793 23846 26433 83279 50288 41971 69399 37510..

Untuk bilangan:

= 1,4142135623730950488016887242096.... atau

= 1,41421 35623 73095 04880 16887 24209 69807 85696 71875 37694 80731 76679 73798..

dan untuk bilangan e:

= 2,7182818....
