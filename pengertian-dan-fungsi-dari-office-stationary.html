6. sebutkan fungsi-fungsi pada office botton


New Digunakan untuk membuat file dokumen kosong baru. Open Icon ini digunakan untuk membuka file dokumen yang tersimpan. Save Sebuah perintah untuk menyimpan file dokumen aktif. Save As Seperti fungsinya sama seperti icon "Save" diatas digunakan untuk menyimpan file, akan tetapi save as digunakan lebih mempunyai fungsi khusus yaitu menyimpan dokumen ke dalam format tertentu. Print Digunakan untuk mencetak file dokumen yang sedang aktif. Prepare Digunakan untuk melakukan properti khusus terhadap file dokumen aktif. Send Digunakan untuk mengirimkan file dokumen aktif sebagai email / faksimile. Publish Perintah untuk menerbitkan dokumen aktif ke website / blog / dll. Close Digunakan untuk menutup file dokumen aktif. Word Option Menutup aplikasi word . Exit Word Tombol untuk merekayasa fitur-fitur penting didalam ms word seperti mengubah fitur tampilan lingkungan word, kustomisasi quick access toolbar, kustomisasi saving file, kustomisasi setting file dll. 16. Fungsi office button pada Microsoft Office Word 2007 mirip dengan fungsi salah satu menu pada Microsoft Office Word 2003 yaitu menu


Fungsi office button pada Microsoft Office Word 2007 mirip dengan fungsi salah satu menu pada Microsoft Office Word 2003 yaitu menu File. Pada microsoft Office Word 2007, seperti kita ketahui tidak terdapat menu File. Seperti kita ketahui, pada microsoft word 2003, sedangkan menu file pada microsoft Word 2007 digantikan dengan Ofiice Button. Pada Office Button sendiri terdapat menu file. Di dalam menu file sendiri terdapat beberapa menu seperti New, Save, Save As, hingga Open. Tidak hanya pada microsoft word 2007 saja, untuk aplikasi Excel 2007 dan Power Point 2007 juga sudah dilengkapi dengan adanya Office Button.

Demikian pembahasan mengenai fungsi Office Button pada Microsoft Word 2007, untuk Office button sendiri memiliki fungsi untuk Save, Save As, Open dan menu lainnya yang terdapat pada menu File pada versi Microsoft Office Word 2003. 32. Apa pengertian dan manfaat dari virtual office?


Virtual office adalah nama lain dari perkantoran maya (yang ada di dunia maya atau di internet). 

Manfaat dari Virtual Office
1. Efesiensi waktu ; jelas dengan adanya kantor virtual / virtual office kita tidak terpaku bekerja di satu tempat dengan jam-jam tertentu, sehingga mengurangi waktu yang tidak begitu perlu seperti waktu untuk perjalanan menuju ke kantor, dan lain sebagainya.
2. Efisiensi tempat ; dengan virtual office anda tidak perlu memerlukan tempat untuk menjalankan bisnis anda, karena semuanya sudah tertata melalui virtual office.
3. Menghemat Biaya Pengeluaran ; yang jelas ini yang menjadi keunggulan dan manfaat yang sangat terasa bagi para pebisnis yang menjalankan bisnisnya melalui virtual office, karena dengan begitu kita bisa mengehemat biaya gedung (sewa/milik sendiri), maintentence, listrik, tranport dan biaya-biaya lain yang ada pada kantor konvensional yang real/nyata.
