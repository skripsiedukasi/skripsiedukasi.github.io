3. apa perbedaan NTFS dan SWAP


NTFS adalah file system yang digunakan pada windows berbasis NT (NT, 2000, XP, 2003, Vista). Pada file system ini besar partisi max 256 Terra Byte sedangkan besar datanya 16 Terra Byte. NTFS support terhadap metadata, yaitu database yang berisi informasi suatu file. Selain itu juga NTFS juga memiliki fasilitas seperti :quota = Pembatasan besar data untuk setiap userenkripsi = Fasilitas proteksi data dengan cara mengacak bit dalam suatu file sehingga tidak bisa terbaca oleh user yang tidak berhakkompresi = Fasilitas pemampatan data sehingga space akan lebih lapang


Swap adalah File System yang tidak digunakan sebagai tempat penyimpanan data, tetapi sebagai virtual memory, yaitu sebagai pembantu kinerja dari si memory. Virtual memory ini juga digunakan pada windows dengan nama page file, tetapi kalau swap ditaruh pada partisi yang berbeda dengan system dan diberi tempat tersendiri, pada page file dia berada pada partisi yang sama dengan system atau data.

4. apa yang dimaksud dengan NTFS file sistem dan FAT file sistem


NTFS adalah sistem file yang lebih disukai untuk versi Windows.
NTFFAT32, dan FAT kurang digunakan, digunakan dalam versi sebelumnya dari sistem operasi Windows, termasuk Windows 95, Windows 98, dan Windows Millennium Edition.

31. - Apakah yang ada ketahui tentang RAID ? - Jelaskan pendapat Anda tentang FAT dan NTFS dalam File system? - Jelaskan Perkembangan Media Penyimpanan dari Awal sampai sekarang? - Jelaskan kaitan antara sistem berkas dengan aplikasi program yang digunakan ?


1. RAID (Redundant Array of Independent Disks) adalah suatu teknologi yang digunakan untuk meningkatkan performa, kehandalan, atau kapasitas penyimpanan suatu sistem komputer dengan menggabungkan beberapa disk drive menjadi satu entitas logis. Ada beberapa tipe RAID yang umum digunakan, seperti RAID 0, RAID 1, dan RAID 5.

2. FAT (File Allocation Table) adalah file system yang digunakan pada sistem operasi Windows sebelum Windows NT. Ini adalah file system yang sederhana dan mudah digunakan, tetapi memiliki keterbatasan dalam hal ukuran file dan jumlah partisi. NTFS (New Technology File System) adalah file system yang digunakan pada sistem operasi Windows NT dan yang lebih baru. Ini lebih canggih daripada FAT dan memiliki fitur seperti keamanan file yang lebih baik, kompresi file, dan dukungan untuk file yang besar.

3. Perkembangan media penyimpanan dari awal sampai sekarang sangat pesat. Pada awalnya, data disimpan pada tape magnetic, disk drive mekanis, dan disket. Kemudian, perkembangan teknologi memungkinkan penggunaan media penyimpanan seperti hard disk drive, flash drive, dan solid state drive. Saat ini, media penyimpanan yang sedang digunakan adalah berbasis cloud, flash disk, dan solid state drive.

4. Sistem berkas adalah sebuah sistem yang digunakan untuk menyimpan, mengelola, dan mengakses data pada suatu komputer. Aplikasi program digunakan untuk mengakses dan mengelola data yang disimpan dalam sistem berkas. Aplikasi program ini mengirim perintah ke sistem berkas untuk mengambil atau menyimpan data, dan sistem berkas mengeksekusi perintah tersebut. Sistem berkas dan aplikasi program saling berkaitan dan saling tergantung satu sama lain dalam proses pengelolaan data.


32. apakah kelebihan dan kekurangan dari Ext4 dan NTFS.....?


Ext4 dirilis secara komplit dan stabil berawal dari kernel 2.6.28 jadi apabila distro anda yang secara default memiliki versi kernel tersebuat atau di atas nya otomatis system anda sudah support ext4 (dengan catatan sudah di include kedalam kernelnya) selain itu versi e2fsprogs harus mengunakan versi 1.41.5 atau lebih.
Apabila anda masih menggunakan fs ext3 dapat mengkonversi ke ext4 dengan beberapa langkah yang tidak terlalu rumit.
Keuntungan yang bisa didapat dengan mengupgrade filesystem ke ext4 dibanding ext3 adalah mempunyai pengalamatan 48-bit block yang artinya dia akan mempunyai 1EB = 1,048,576 TB ukuran maksimum filesystem dengan 16 TB untuk maksimum file size nya,Fast fsck,Journal checksumming,Defragmentation support.

NTFS di kenalkan pertama pada Windows NT dan merupakan file system yang benar benar berbeda di banding teknologi FAT. NTFS menawarkan security yang jauh lebih baik, kompresi file, cluster dan bahkan support enkripsi data. NTFS merupakan file system standar untuk Windows Xp dan apabila anda melakukan upgrade Windows biasa anda akan di tanyakan apakah ingin mengupgrade ke NTFS atau tetap menggunakan FAT. Namun jika anda sudah melakukan upgrade pada Windows Xp dan tidak melakukan perubahan NTFS itu bukan masalah karena anda bisa mengkonversinya ke NTFS kapanpun. Namun ingat bahwa apabila anda sudah menggunakan NTFS akan muncul masalah jika ingin downgrade ke FAT tanpa kehilangan data.
Pada Umumnya NTFS tidak kompatibel dengan Operating System lain yang terinstall di komputer yang sama (Double OS) bahkan juga tidak terdeteksi apabila anda melakukan startup-boot menggunakan floopy. Untuk itu sangat disa-rankan kepada anda untuk menyediakan partisi yang kecil saja yang menggunakan file system FAT di awal partisi. Partisi ini dapat anda gunakan untuk menyimpan Recovery Tool apabila mendapat masalah.
