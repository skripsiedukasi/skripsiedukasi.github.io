1. jelaskan pengertian start pendek, menengah dan start panjang! ​


Start Pendek (Bunch Start)

Start pendek ini dilakukan dengan cara berjongkok, dengan kaki kiri di depan dan lutut kaki kanan diletakkan sejajar di sebelah kaki kiri, beri jarak sekitar satu kepal. Jari-jari tangan rapat dan ibu jari terpisah, keduanya diletakkan di belakang garis start.

Start menengah (medium start)

Lutut kaki kanan menempel di tanah. Kaki kiri berada di samping lutut kaki kanan dengan jarak sekitar satu kepal. Kedua tangan menempel di atas garis start dengan membentuk huruf "V" terbalik. Pandangan rileks ke depan, konsentrasi pada aba-aba start berikutnya

Start panjang (long start)

Lutut kaki kanan menempel di tanah. Kaki kiri berada di depan dengan posisi jinjit. Letak ujung kaki belakang segaris dengan tumit kaki depan adalah sikap start jongkok panjang. Kedua tangan menempel di atas garis start dengan membentuk huruf "V" terbalik


2. jelas pengertian start pada lari jarak pendek!


Teknik Start Start adalah sikap dan gerakan awal untuk memulai lari, start dalam nomor lari terbagi menjadi 3 macam, yaitu:
a. Start berdiri (standing start) start berdiri digunakan untuk lari jarak menengah dan lari jarak jauh. 
b. Start melayang (flying start) start melayang digunakan oleh pelari II, III, dan IV dalam lari estafet 4 x 100meter 
c. Start Jongkok (cruched start) start jongkok digunakan dalam lari jarak pendek.Start merupakan sikap awalan pada saat waktu yang akan melakukan jalan atau lari dengan kaki atau tangan kita tapi tidak boleh terkena batas.
Macam-macan start
-star jongkokk merupakan start yang digunakan ketika hendak melakukan lari jarak pendek (100m) yang biasa kita sebut lari cepat.
-Start melayang merupakan start yang digunakan lari jarak menengah ,khususnya pada lari sambung atau estafet terutama pada pemain kedua,ketiga dan seterusnnya.
-Start Berdiri digunakan lari jarak jauh seperti pada lari maraton dan lintas alam.

10. Tugas jelaskan cara melakukan start pendek ,(bunch start) start menengah (medium start ) dan start panjang (long start ) ​


Berikut adalah cara pelaksanaan dari short start, medium start dan long start untuk cabang lari jarak pendek.

Short Start :

Kaki kanan di depan, kaki kiri di belakang. Jari-jari kaki belakang segaris dengan tumit kaki depan

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Medium Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan ujung jari

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Long Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan tumit

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

13. Sebutkan dan jelaskan aba aba melakukan start jongkok/start pendek dalam perlombaan dari jarak pendek


Jawaban:

Untuk menjadi pelari yang handal membutuhkan penguasaan teknik start yang benar. Salah satunya adalah posisi start jongkok. Berikut adalah aba-aba dan posisi start jongkok :

Aba-aba "bersedia"

Posisi yang dilakukan pada aba-aba ini adalah

-badan berdiri lurus dan tegak pada 3 langkah belakang garis start

-saat aba-aba disuarakan maju dan letakkan kaki di belakang pas garis start dan letakkan lutut tungkai sejajar dengan kaki

-kedua tangan berada di belakang garis

-pandangan melihat ke depan dan leher sebaiknya rileks tidak kaku

Aba-aba "siap"

Posisi yang dilakukan pada aba-aba ini adalah :

-mengangkat pinggul ke atas

-lutut kaki dengan posisi sudut 90° dan tungkai 110°-120°

-pandangan mukai berubah mendekat garis start

-mendorong pundak ke depan dengan tumpuan lengan

Aba-aba "ya"

Posisi yang dilakukan pada aba-aba ini adalah :

- mengayunkan lengan dan kaki dengan kuat

- meluruskan badan dengan posisi condong ke depan

-kaki belakang digerakkan ke depan dengan secepat-cepatnya

- mendorong tungkai belakang maju ke depan sekuat mungkin

-melakukan start dengan secepatnya

Penjelasan:

Start ialah merupakan suatu persiapan awal seorang pelari yang dimana akan melakukan gerakan lari. Tujuan utama start dalam lari jarak pendek, lari estafet, dan lomba lari gawang adalah untuk mengoptimalisasikan pola lari percepatan.

Jawaban:

Kelas: 8

Kata Kunci: start pada lari

Kode: 8.22.3

Pada lari jarak pendek, kita akan menggunakan start jongkok. Dengan menggunakan start jongkok, ledakan tenaga yang dihasilkan untuk lari akan lebih besar daripada start berdiri pada lari jarak menengah, yang mana tenaga besar tersebut memang diperlukan untuk lari dengan jarak yang pendek. Namun sebaliknya, start lari jarak menengah menggunakan start berdiri yang tidak memberikan ledakan tenaga sebesar start jongkok, yang mana memang baik untuk lari jarak menengah karena kita harus menyimpan tenaga untuk melakukan lari jarak menengah.

38. jelaskan perbedaan start menengah dengan start pendek!


START PENDEK (BUNCH START)

Start pendek adalah lutut kaki belakang (kaki kiri) diletakkan didepan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.


START MENENGAH (MEDIUM START)

Start menengah adalah lutut kaki belakang (kaki kiri) diletakkan sejajar dengan ujung kaki depan (kaki kanan) dengan jarak satu kepal jari-jari tangan rapat dan ibu jari terpisah.

Perbedaannya hanya terletak pada posisi lutut dengan telapak kaki.
Apabila start pendek maka posisi lutut sejajar dengan jari-jari kaki
Apabila start menengah maka posisi lutut sejajar dengan tengah telapak kaki
